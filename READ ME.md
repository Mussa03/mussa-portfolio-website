# Mussa Njechelani - Personal Portfolio Website

## 🔧 Tools Used
- HTML5
- CSS3
- JavaScript
- VS Code
- Git & GitHub

## 🌟 Key Features
- Multi-page layout: Home, About, Projects, Contact, Survey
- Dark/Light mode toggle
- Contact form with validation
- Responsive design for mobile & desktop
- Image map and embedded Google Map/YouTube
- DOM manipulation (time-based greeting, toggles)

## 🚧 Challenges Faced
- Mobile navigation toggle: Solved with CSS media queries + JavaScript toggle.
- Validating phone numbers in Tanzanian format using regex.
- Image map resizing: used specific `coords` and `usemap` adjustments.
